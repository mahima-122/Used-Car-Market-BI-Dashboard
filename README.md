Used Car Market: Interactive Business Intelligence Dashboard

## Project Overview
This project focuses on the "Visual Analytics" stage of the data pipeline. Building upon the cleaned dataset from my [Used Car Analysis (Python) project]([(https://github.com/mahima-122/Used-Car-Market-Data-Analysis)]), this dashboard transforms raw metrics into an interactive, high-level business intelligence tool.

## Tech Stack
- **Tool:** Power BI Desktop
- **Data Source:** Cleaned Used Car dataset (Pre-processed via Pandas)
- **Visualization:** Bar Charts, Donut Charts, Trend Analysis, and Interactive Slicers

## Dashboard Insights
The dashboard serves as a decision-making tool for stakeholders:
- **Pricing Strategy:** Visualized total market value by manufacturer, identifying premium vs. mass-market brands.
- **Market Distribution:** Used a donut chart to analyze the ratio of total mileage to total price, highlighting market investment patterns.
- **Inventory Analysis:** Tracked the count of vehicles by fuel type and company, revealing market dominance (e.g., Maruti’s high market share).
- **Interactive Exploration:** Integrated **Fuel Type Slicers** allowing users to drill down into specific vehicle segments (Petrol, Diesel, LPG).

## Key Learning Outcomes
- **Data Modeling:** Transitioned from flat-file data to a relational structure suitable for BI.
- **Visual Storytelling:** Applied design principles (dark-mode aesthetic, card KPIs, and consistent color-coding) to ensure data clarity.
- **Business Reporting:** Developed KPIs that translate complex data into readable metrics for non-technical users.
